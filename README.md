## рҹҡҖ Getting Started

### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/Suman18-bit/Deep-Learning.git
cd Deep-Learning
```

### 2пёҸвғЈ Set Up Environment

```bash
# Create a virtual environment (recommended)
python -m venv dl-env
source dl-env/bin/activate        # Linux/Mac
dl-env\Scripts\activate           # Windows
```

### 3пёҸвғЈ Install Dependencies

```bash
pip install tensorflow torch torchvision keras \
            numpy pandas matplotlib seaborn \
            scikit-learn opencv-python jupyter
```

### 4пёҸвғЈ Launch Jupyter Notebook

```bash
jupyter notebook
```

> вҳҒпёҸ **Prefer cloud?** Open any notebook directly in [Google Colab](https://colab.research.google.com) вҖ” free GPU included!
